# 歌词格式转换工具

一个简单易用的在线歌词格式转换工具，支持 LRC 和 KRC 格式的互相转换。

## 功能特点

- ✨ 支持 LRC 转 KRC
- ✨ 支持 KRC 转 LRC
- 📁 支持文件拖拽上传
- 👀 实时预览转换结果
- 💾 一键下载转换后的文件
- 🎯 简单直观的用户界面


## 使用说明

1. 选择转换类型（LRC 转 KRC 或 KRC 转 LRC）
2. 点击上传区域或将文件拖拽到上传区域
3. 上传完成后点击"转换"按钮
4. 预览转换结果
5. 点击"下载转换结果"保存文件

## 构建部署

```bash
npm run build
# 或
yarn build
```

构建后的文件将生成在 `dist` 目录下。
